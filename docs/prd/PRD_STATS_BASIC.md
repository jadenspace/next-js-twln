# PRD – 로또 기본 통계 분석 페이지

## 1. 개요
전체 회차 또는 사용자가 지정한 특정 회차 범위를 대상으로 로또 번호의 기초적인 통계 데이터를 시각화하여 제공합니다.

## 2. 공통 기능: 회차 검색 (Draw Search)
모든 통계 페이지 상단에는 공통적인 필터 UI가 포함되어야 합니다.

- **전체 조회**: 1회차부터 최신 회차까지 전체 데이터 집계
- **구간 조회**: 시작 회차와 종료 회차를 직접 입력 (예: 1000회~1100회)
- **최근 N회 조회**: 최신 회차 기준 10, 20, 50, 100회 등 프리셋 제공
- **적용 버튼**: 필터 변경 시 해당 조건으로 통계 데이터 재계산 및 차트 갱신

## 3. 상세 메뉴별 요구사항

### 3.1 번호별 통계 (Numbers)
- **목적**: 각 번호(1~45)의 총 출현 횟수 분석
- **표시 데이터**: 번호별 빈도, 평균 출현율 대비 증감
- **시각화**: 1~45번 바 차트 또는 히트맵 (많이 나온 번호 진하게)
- **정렬**: 번호순, 빈도 높은 순, 빈도 낮은 순

### 3.2 구간별 출현횟수 (Ranges)
- **목적**: 번호대(10단위)별 출현 빈도 확인
- **구간 정의**: 1~10, 11~20, 21~30, 31~40, 41~45
- **시각화**: 파이 차트 또는 누적 가로 바 차트
- **인사이트**: 특정 번호대 멸(Myeol, 출현 번호 없음) 패턴 강조

### 3.3 기간별 미출현 번호 (Missing)
- **목적**: 최근 일정 기간 동안 당첨되지 않은 번호 집계
- **계산**: 선택된 회차 범위 내에서 출현 횟수가 0인 번호 추출
- **표시**: 미출현 기간(회차 수)을 역순으로 표시하여 '장기 미출현수' 강조
- **시각화**: 번호 카드 형태 (미출현 회수 병기)

### 3.4 홀짝 통계 (Odd-Even)
- **목적**: 당첨 번호 6개의 홀수와 짝수 비율 분석
- **비율 조합**: 0:6, 1:5, 2:4, 3:3, 4:2, 5:1, 6:0
- **시각화**: 회차별 홀짝 비율 분포 히스토그램
- **인사이트**: 가장 빈번한 조합(주로 3:3, 4:2, 2:4)의 비중 표시

### 3.5 연속번호 출현 (Consecutive)
- **목적**: 한 회차 내에서 인접한 숫자가 동시에 출현한 빈도 분석
- **연속쌍 정의**: (1, 2), (10, 11) 등
- **표시 데이터**: 2연번, 3연번 이상 출현 빈도, 연속쌍 미출현 회차 비중
- **시각화**: 회차별 연번 포함 여부 타임라인 차트

## 4. 기술적 요구사항
- **데이터 로드**: 초기 로드 시 최신 회차 기준 전체 통계 표시
- **성능**: 대량의 회차 데이터 처리 시 클라이언트 측 캐싱 활용
- **반응형**: 모바일 환경에서 차트가 잘리지 않도록 가로 스크롤 또는 유동적 레이아웃 적용
